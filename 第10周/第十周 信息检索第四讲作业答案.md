## 习题4-1

给定:
文档数量:800000
每个文档平均词项数:200
总词项数:400万
磁盘平均寻道时间:5ms
则:

1. T = 文档数量 x 平均词项数 = 800000 x 200 = 160,000,000
2. 比较次数 = T log2T = 160,000,000 x log2 160,000,000 ≈ 160,000,000 x 25 = 4,000,000,000
3. 每次比较需2次磁盘寻道,所以总寻道次数= 2 x 4,000,000,000 = 8,000,000,000
4. 总时间 = 磁盘寻道时间 x 磁盘寻道次数 
          = 5ms x 8,000,000,000 
          = 40,000,000,000ms
          = 40,000,000s
          = 666,667分钟
          = 11,111小时
          = 463天
