> `strcmp(token, table[i] == 0)`
>
> `"bool" 类型的实参与 "const char *" 类型的形参不兼容C/C++(167)`

```cpp
char reserve(char token[])
{
    const char *table[] = {"begin", "end", "integer", "real"};
    const char code[] = "{}ac";
    for (unsigned i = 0; i < strlen(code); i++)
    {
        if (strcmp(token, table[i] == 0))
            return code[i];
    }
    return 'i';
}
```
